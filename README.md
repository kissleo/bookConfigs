---
#### <br>如有失效或者有想做成书源的网站可以发issue或者群里@我

* 导入书源的三种种方式:
	* 1.一键导入([点击导入](https://xgg.kim/t5N83))
	* 2.复制括号内的内容打开爱阅书香-右上角书源配置-同步-粘贴-获取书源配置(https://github.com/gangxiaoji/bookConfigs)
	* 3.下载书源后选择使用爱阅打开(点击书源名可下载)
